# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

In the spirit of mental safety and data privacy, please report any security vulnerabilities via GitHub Issues or by contacting the maintainer directly.

### Our Commitment
- **PII Protection**: We use automated guardrails (The Guardian agent) to detect and mask Personally Identifiable Information in transcripts.
- **Local-First History**: Session data is sharded and stored locally by default to ensure maximum user control.
- **Non-Diagnostic**: The agent is strictly designed for "Chaos Reduction" and organizational micro-actions. It refuses to provide medical, legal, or financial advice.
